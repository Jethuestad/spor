# Utilities (React)

A set of utilities to build wonderful apps with.

## Installation

```bash
$ npm install @vygruppen/spor-util-react
```

## Usage

```tsx
import {
  useBreakpointValue,
  useClipboard,
  useControllableProp,
  useDisclosure,
  useMediaQuery,
  useMergeRefs,
  useOutsideClick,
  usePrefersReducedMotion,
  useTheme,
  useToken,
} from "@vygruppen/spor-util-react";
```
Please refer to the docs for more information about usage.
