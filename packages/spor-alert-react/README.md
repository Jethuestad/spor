# Alert (React)

This package contains alert components.

## Installation

```bash
$ npm install @vygruppen/spor-alert-react
```

## Usage

```tsx
import {
  StaticAlert,
  ClosableAlert,
  ExpandableAlert,
} from "@vygruppen/spor-alert-react";
```

Please refer to the documentation for more information.
