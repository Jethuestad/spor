# Loader (React)

This package contains skeletons and loading animations used in Spor.

## Installation

```bash
$ npm install @vygruppen/spor-loader-react
```

## Usage

```tsx
import {
  Skeleton,
  SkeletonCircle,
  SkeletonText,
  Spinner,
  ContentLoading,
  InlineLoading,
  FullScreenLoading,
} from "@vygruppen/spor-loader-react";
```

Please refer to the documentation for further details.
