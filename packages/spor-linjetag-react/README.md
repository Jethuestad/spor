# Linjetag (React)

This package contains linjetags for React.

## Installation

```bash
$ npm install @vygruppen/spor-linjetag-react
```

## Usage

```tsx
import { TravelTag, InfoTag, LineIcon } from "@vygruppen/spor-linjetag-react";
```

Please refer to the documentation site for usage.
